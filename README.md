# Portfolio Website

A modern, responsive portfolio website showcasing my skills, projects, and experience as a Full-Stack Software Developer.

**[View the live application here](https://olivertaylor-dev.vercel.app/)**

## Features

- Responsive single-page application with smooth scrolling navigation
- Home section with animated skill display using react-typed
- About section highlighting background and experience
- Skills section with organized technology display
  - Compact card grid on desktop/tablet
  - Pill-style layout optimized for mobile
- Project showcase featuring:
  - No Dice Games (Board Game Review Website)
  - FanFinder (Location-based Social Networking App)
  - Weather App
  - Vocabulary Learning Game
  - Text-based RPG (in development)
- Detailed project pages with tech stack information and live demos
- Contact section for getting in touch
- Scroll-triggered reveal animations

## Tech Stack

- **React**
- **Tailwind CSS**
- **React Scroll**
- **React Typed**
- **React Icons**
